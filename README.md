탄소 다이어트 앱(carbon_diet_app)

1. 프로젝트 개요

일상생활(식사, 교통, 전기)에서 발생하는 탄소 배출량을 기록하고 관리하는 모바일 애플리케이션

2. 개발 동기 및 목적

기후 변화에 대한 생각이 유튜브를 보면서 되게 심각하다고 생각해서 개인의 탄소 배출량 관리가 중요하다는 것을 느끼게 되었다. 사용자들이 쉽게 자신의 탄소 배출량을 관리할 수 있는 도구를 제공하여, 환경 보호에 대한 인식을 높이고 실천을 독려하는 것을 목표로 한다.

3. 사용 기술
- Frontend: Flutter
  
- Backend: Spring Boot
  
- Database: PostgreSQL


4. 주요 기능

4.1 사용자 인증
- 로그인 및 회원가입 기능
- 비밀번호 해싱을 통한 보안 강화
- 사용자 정보를 안전하게 데이터베이스에 저장

4.2 탄소 배출량 기록 및 계산
- 세 가지 카테고리별 기록 가능:
  - 식사: 식단 유형에 따른 탄소 배출량 계산
  - 교통: 이동 수단 및 거리 기반 배출량 계산
  - 전기: 전력 사용량 기반 배출량 계산

4.3 통계 및 시각화
- 주간 배출 패턴 그래프
- 달력 뷰를 통한 일별 배출량 확인

4.4 환경 정보 매거진
- 기후 변화 및 환경 관련 뉴스 제공
- 탄소 중립 실천 방법 안내
- 환경 보호 팁 및 정보 공유

4.5 개인 설정
- 사용자 프로필 관리
- 알림 설정
- 앱 사용 환경 설정

5. 화면 구성

로그인 화면
- 기존 사용자 로그인 인터페이스
- 회원가입 화면으로 이동 가능

회원가입 화면
- 신규 사용자 정보 입력
- 비밀번호 암호화 처리 후 DB 저장

홈 화면
- 오늘의 탄소 배출량 요약
- 주요 기능 바로가기

통계 화면
- 달력 형식으로 일별 배출량 표시
- 날짜 선택 시 해당일의 상세 배출 내역 확인
- 더블 클릭으로 새로운 기록 추가 가능

기록 추가 화면
- 플러스 버튼을 통한 접근
- 전기/교통/식사 카테고리 선택
- 각 카테고리별 상세 정보 입력

매거진 화면
- 환경 관련 기사 및 정보 목록
- 기사 클릭 시 상세 내용 표시

설정 화면
- 사용자 정보 수정
- 앱 환경 설정
